# Frontend AI Foundation

This repository was created for the **FlyRank Frontend AI Engineering** track.

## Project Goal

The goal of this repository is to practice AI-assisted frontend development, Git workflows, and modern engineering conventions.

## Tech Stack

- HTML5
- CSS3
- JavaScript (ES6+)
- React
- Node.js
- Git
- GitHub

## Development Principles

- Use semantic HTML and accessible UI patterns
- Write clean and maintainable code
- Prefer reusable components
- Use modern JavaScript syntax
- Follow Conventional Commits
- Use AI tools to review and improve project documentation

## AI-Assisted Development

Claude Code is used as an AI engineering assistant to review project files, suggest improvements, and support frontend development decisions.
