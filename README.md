# Frontend AI Foundation

This repository was created for the **FlyRank Frontend AI Engineering** track.

## Project Goal

The goal of this repository is to practice AI-assisted frontend development, Git workflows, and modern engineering conventions.

## Getting Started

1. **Clone the repository** to your local machine.
2. **Read `CLAUDE.md`** — it defines project conventions and instructions for AI coding assistants working in this repo.
3. **Build with AI assistance** — use Claude Code or another AI tool to scaffold frontend features, review code, and improve documentation.
4. **Commit with Conventional Commits** — prefix messages with types like `feat:`, `fix:`, or `docs:` (see examples in `CLAUDE.md`).

This repository starts with documentation only. Your first tasks are typically to add project structure (for example, HTML/CSS/JS or a React app) and iterate using the development principles below.

## Tech Stack

- HTML5
- CSS3
- JavaScript (ES6+)
- React
- Node.js
- Git
- GitHub

## Development Principles

- Use semantic HTML and accessible UI patterns
- Write clean and maintainable code
- Prefer reusable components
- Use modern JavaScript syntax
- Follow Conventional Commits
- Use AI tools to review and improve project documentation

## AI-Assisted Development

Claude Code is used as an AI engineering assistant to review project files, suggest improvements, and support frontend development decisions.
